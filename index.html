关于行程A和行程B

发表于 2017-06-11

此文由群管理 江苏太阳1.4臻享白 整理分享

其实这两个行程如果没有清过零，作用就是一样的，但之所以设两个行程，只是为了方便单独计算。打个比方，你今天上高速了，想单独计算高速这个路况下的油耗，那么，在高速起始段就可以将行程A或行程B中的一个单独清零，那么，当你跑完这段路程后，再看当时被你清零的A或B行程的数据，就是这段高速下来的综合油耗。

这里要提示一下的是，最好你清零时选择了A或B后，以后就一直以当初那个被清零的行程来再次清零，而不要一会清零A，一会清零B，这样做的目的，是让其中一个行程始终保留全面的油耗数据。比如我现在就是清零了B行程，但A行程始终保留不清零。

这么说，大家对行程AB的作用应该就明白了吧。

车载大屏自动开启

发表于 2017-06-09

此文由群管理 江苏太阳1.4臻享白 整理分享

关于中控大屏自动开启的情况，分为:

  1. 如果你熄火之前，车机大屏没有关闭，则下次再次点火时，大屏会自动开启。

  2. 如果在临时熄火停车后，隔不长时间后，再次点火时，不管你上次熄火时是否先闭没关闭车机大屏，都会自动开启大屏。

  1. 如果插上U盘，并有U盘自动播放，则会自动开启大屏。

其它自动开启情况，则需要再行验证，本人只验证到这些状况下了。

高速省油的方式

发表于 2017-06-07

此文由群管理 江苏太阳1.4臻享白 整理分享

因为部分车友对油耗比较在意，所以专门说下省油的方法:

  1. 油耗单位请设置为L/100KM。这是我们经常说你跑了几个油几个油时用的单位。

  2. 市区路况一般省油效果有限，因为怠速运行太多，所以，这里说的是如何在高速上省油。

  3. 起速要快，越快脱离低速区，越利于省油。要点是油门控制果断，利用加减油门时机快速升档。

  4. 起速以后，一般指三档以后。油门控制则应该缓轻，不要深一脚浅一脚地踩油门。保持车速均匀增加和减少。

  5. 少踩刹车。预判前方有情况时，就提前松油，尽量使车辆自然减速后再刹车。

  1. 多用D档滑行，自动档车只有在带档滑行时瞬时油耗才是零。不要空档滑行。

  2. 不是越快越省油，而是保持经济时速，区间为80码上下20码的样子。到了上限，就松油门让车滑行，到了下限则均匀加油，让速度提至上限。

  3. 尽量少开窗，减少风阻，如果想透气，可把天窗开至经济模式。如果非要开窗不可，也采用对角开启方式。如主驾对应后排右侧

  4. 最后，对于经验丰富的车主，更可利用手动模式更细化在各速度区间的档位加减配合。由于这里牵涉经验问题，新手如果不知何时该加减，油门不会配合，反而适得其反，所以，就不多说了。

安全驾驶，人人有责。祝各位行车平安，高效。

指南者车灯延时关闭(伴我回家)功能的实现

发表于 2017-05-10

此文由群管理 江苏太阳1.4臻享白 整理分享

如果要激活车灯延时关闭功能，需要在车灯还未关闭时，首先按一键启动键将车辆熄火。(钥匙式开关则转到stop位置)。然后，在2分钟时间之内，将车灯开关旋转至关闭位置。此时，车灯进入90秒的延时，90秒后，车灯会自动熄灭。

对于开启了自动大灯功能的车型，如果大灯在车辆熄火前处于照明状态。则只需按一键启动键将车辆熄火即可，车灯会自动延时90秒关闭。

注意，如果不是自动大灯，或没有打开自动大灯功能的车辆，熄火后不把灯光开关置于关闭状态，或超过2分钟后才将车灯开关置于关闭位置，或关闭灯光开关后再次接通了电源。延时关闭功能都将失效。

指南者音乐与导航语音兼容问题

发表于 2017-05-06

此文由群管理 江苏太阳1.4臻享白 整理分享

本人用的是安卓三星机，不是苹果。结果只对安卓有效。

今天又试验了手机与车机互联的情况:

  1. 单连usb，正常充电，不开车机，高德，凯立德语音正常从手机发出。

  2. 连usb，同时打开车机，正常充电，放音乐也正常，导航语音也从手机发出。

  3. 在2的情况下，如果同时打开蓝牙，则充电正常，但音乐与导航会出现干扰，就是如果用USB放音乐，音乐正常播放，但手机端导航会无声。如果用蓝牙放音乐，则音乐会在导航播报时暂停，但导航语音却无法听到。

  4. 断开USB，只用蓝牙连接手机与车机，则依然会出现音乐与导航语音不兼容情况。

  5. 开蓝牙时，将手机与车机的uconnect live连接上，用酷我音乐放音乐，则音乐正常播放，导航语音从车载喇叭发声。互不干扰。

  6. 在5的基础上，连上USB正常充电。没有问题。

  • 总结:

    1. 出现导航语音与音乐干扰情况，是在蓝牙连接时出现，如果又想充电，又想听导航，请关闭手机上的蓝牙连接。

    2. USB单连时，歌曲从车机发声，导航从手机发声，不干扰。

    3. USB充电任何时候都正常。

    4. 开蓝牙连接时，只有通过与uconnect live连接，才能使音乐与导航语音共同从车载喇叭发声。
