关于行程A和行程B

发表于 2017-06-11

此文由群管理 江苏太阳1.4臻享白 整理分享

其实这两个行程如果没有清过零，作用就是一样的，但之所以设两个行程，只是为了方便单独计算。打个比方，你今天上高速了，想单独计算高速这个路况下的油耗，那么，在高速起始段就可以将行程A或行程B中的一个单独清零，那么，当你跑完这段路程后，再看当时被你清零的A或B行程的数据，就是这段高速下来的综合油耗。

这里要提示一下的是，最好你清零时选择了A或B后，以后就一直以当初那个被清零的行程来再次清零，而不要一会清零A，一会清零B，这样做的目的，是让其中一个行程始终保留全面的油耗数据。比如我现在就是清零了B行程，但A行程始终保留不清零。

这么说，大家对行程AB的作用应该就明白了吧。

车载大屏自动开启

发表于 2017-06-09

此文由群管理 江苏太阳1.4臻享白 整理分享

关于中控大屏自动开启的情况，分为:

  1. 如果你熄火之前，车机大屏没有关闭，则下次再次点火时，大屏会自动开启。

  2. 如果在临时熄火停车后，隔不长时间后，再次点火时，不管你上次熄火时是否先闭没关闭车机大屏，都会自动开启大屏。

  1. 如果插上U盘，并有U盘自动播放，则会自动开启大屏。

其它自动开启情况，则需要再行验证，本人只验证到这些状况下了。

高速省油的方式

发表于 2017-06-07

此文由群管理 江苏太阳1.4臻享白 整理分享

因为部分车友对油耗比较在意，所以专门说下省油的方法:

  1. 油耗单位请设置为L/100KM。这是我们经常说你跑了几个油几个油时用的单位。

  2. 市区路况一般省油效果有限，因为怠速运行太多，所以，这里说的是如何在高速上省油。

  3. 起速要快，越快脱离低速区，越利于省油。要点是油门控制果断，利用加减油门时机快速升档。

  4. 起速以后，一般指三档以后。油门控制则应该缓轻，不要深一脚浅一脚地踩油门。保持车速均匀增加和减少。

  5. 少踩刹车。预判前方有情况时，就提前松油，尽量使车辆自然减速后再刹车。

  1. 多用D档滑行，自动档车只有在带档滑行时瞬时油耗才是零。不要空档滑行。

  2. 不是越快越省油，而是保持经济时速，区间为80码上下20码的样子。到了上限，就松油门让车滑行，到了下限则均匀加油，让速度提至上限。

  3. 尽量少开窗，减少风阻，如果想透气，可把天窗开至经济模式。如果非要开窗不可，也采用对角开启方式。如主驾对应后排右侧

  4. 最后，对于经验丰富的车主，更可利用手动模式更细化在各速度区间的档位加减配合。由于这里牵涉经验问题，新手如果不知何时该加减，油门不会配合，反而适得其反，所以，就不多说了。

安全驾驶，人人有责。祝各位行车平安，高效。

车身清洗之水膜洗车法

发表于 2017-05-31

此文由群管理 江苏太阳1.4臻享白 整理分享

关于洗车，先说一下，如果你不是很在乎漆面维护，或者你比较懒，一切相关事项都交给洗车行负责的话，那么就不用看这篇文章了。

首先，不要用加压水流洗车，这个很多人不理解，其实，洗车行用加压水枪的目的，只是因为快，并不是洗得好。加压水流只能冲走一些较大块的附着物，但是一些被静电吸附在车身上的细小灰砂，油膜等是冲不走的。

正确保护漆面的洗车流程是什么样的:

用一般自来水接上水管，先对汽车来个全面冲湿。如果有较大的表面污物，一般就先冲走了。
然后准备好洗车水蜡，海锦之类的洗车用品，水蜡初次加水稀释不要太稀，用海绵充分在水蜡溶液中浸透。
此时调节水管的流量也不用太大，以能充分在车身上形成水膜，但又不产生很大水溅为准。这里说一下水膜洗车的好处。
只有压力合适的水，才可以在车身上形成水膜，水膜的表面张力，可以在一定程度上隔绝附着在车身上的砂粒对漆面的划伤。这是高压水流无法做到的。
好了，接下来就是，一手拿着充分吸收了水蜡溶液的海绵。一手拿着调好压力的自来水管，将水从上向下柔和地流在车身上，当水膜在车身表面形成水膜后，用海绵顺水流动方向，轻轻擦一下车身，由于水膜的隔离，加上海绵水蜡的分解与润滑作用，很轻易就能把附着在车身上的细小污带清理掉，而且最大程度减小灰砂对漆面的划伤，水膜形成到哪洗到哪。如果海绵溶液太稀了，就再次充分吸取水腊后再行擦洗。
全车洗完，如果有更高要求的，可以再洗一下。因为前面水蜡稀释不够稀，，这时应该还有一部分溶液，再加点水即可。海绵充分吸取后，把车身全部用泡沫涂满，另外对一些细节部位着重处理一下。
最后开大水量，对车身第二次全面冲洗清干净即可。
当然，完事后，对轮胎区域也可以仔细冲一下，因为轮胎无漆面，这个就不讲究，清洗干净即可。
擦干车身各部位。等待干透以后，你想打蜡什么的。自己喜好了。
这种水膜洗车法，是最护漆面又干净的方法。另外，擦洗环节，注意海绵直线向下运动，不要在漆面上划圈。

为什么不能空档滑行

发表于 2017-05-30

此文由群管理 江苏太阳1.4臻享白 整理分享

再来谈一下为什么不建议空档滑行。

首先一点，任何车型，包括手动档，都不建议空档滑行，老司机有一些恶习，但不代表就是正确的。

空档滑行并不省油

对于老式手动档车型来说，空档是让发动机强制进入怠速状态。但发动机依然会喷油维持运行，只是相对于挂载状态，喷油量会少一点，所以会有轻微节油效果。但是，现在的家用车，已普遍采用电脑控制的电喷系统。这个系统，有减速减油的逻辑，如果你把档位挂入空档，则电脑会命令喷油嘴怠速喷油以维持发动机运转。而如果你把档位停留于D档，来自于车轮的转动反馈力，依然可以带动发动机运转，此时电脑则会停止让喷油嘴喷油。这个，各位可以在汽车时速拉起来后，然后松开油门，观察仪表盘的瞬时油耗，一定可以看到油耗为0。所以，空档依然喷油，而带档不喷油，你还认为空档能省油吗?

空档滑行并不能保护双离合

这个多次说过，双离合发热风险，只来自于低速区频繁换档的蠕行状态下，而汽车正常行驶状态下，切换档位并不频繁，加上行驶中的空气流动，足够离合器散热，此时多此一举来个空档滑行，并不能对双离合有任何保护作用。

空档滑行中如果换档，反而会严重损坏变速箱

一旦行驶中切入空档，则不要再试图挂回D档或R档，因为此时因为发动机与变速箱动力已经断开后，双方的转速已不匹配。也就是存在转速差，此时强行挂入D档或R档，则双方转速差的冲力，完全会由啮合齿轮来承受，转速差越大，齿轮受到的冲力也越大。轻则造成啮合位置不正确，导致触发系统传感器报警。重则造成齿轮打齿报废，变速箱直接大修的严重后果。

空档滑行导致发动机制动失去作用，严重影响行车安全

可能大家以为，行驶中最重要的制动力来自刹车，其实大错特错，行驶中，最重要的制动力其实来自于发动机减速制动的效果，带档滑行时，当踩下刹车时，电脑会在刹车优先系统的作用下，强制发动机减速运行，而发动机的减速又通过离合器，变速箱，传动轴反馈至车轮上，带动车轮产生减速。此时再配合刹车盘的辅助作用，才能安全地制动停车。但是，假如空档滑行状态，发动机制动力无法传导至车轮，此时车轮减速完全靠刹车盘来完成。很容易因为刹车过热而导致刹车失效。引发严重安全问题。

空档滑行时，ESP及制动系统效能都大打折扣

ESP，防侧溜稳定车身系统，必须依靠在侧滑产生时，分配各个车轮的动力与制动力大小来实现，一旦你进入空档滑行状态，发生侧滑时，ESP将无法控制对车轮的动力补偿，因为发动机被空档了，动力根本传不过来。这样侧溜乃至翻车的风险就成倍提高。尤其是对四驱车型，更是危险。另外，由于空档，制动系统的液压助力会大大减弱，刹车力度亦大大减小。

下坡空档如同自杀

这个就不多说了，记住这句话就好。

所以，综上所述，对于现在的电喷汽车来说，空档滑行不省油还费油，对双离合又没有实质性保护作用，同时要承受相关系统失灵或损害风险，这样的危险操作，你认为有必要吗?

指南者车灯延时关闭(伴我回家)功能的实现

发表于 2017-05-10

此文由群管理 江苏太阳1.4臻享白 整理分享

如果要激活车灯延时关闭功能，需要在车灯还未关闭时，首先按一键启动键将车辆熄火。(钥匙式开关则转到stop位置)。然后，在2分钟时间之内，将车灯开关旋转至关闭位置。此时，车灯进入90秒的延时，90秒后，车灯会自动熄灭。

对于开启了自动大灯功能的车型，如果大灯在车辆熄火前处于照明状态。则只需按一键启动键将车辆熄火即可，车灯会自动延时90秒关闭。

注意，如果不是自动大灯，或没有打开自动大灯功能的车辆，熄火后不把灯光开关置于关闭状态，或超过2分钟后才将车灯开关置于关闭位置，或关闭灯光开关后再次接通了电源。延时关闭功能都将失效。

指南者音乐与导航语音兼容问题

发表于 2017-05-06

此文由群管理 江苏太阳1.4臻享白 整理分享

本人用的是安卓三星机，不是苹果。结果只对安卓有效。

今天又试验了手机与车机互联的情况:

  1. 单连usb，正常充电，不开车机，高德，凯立德语音正常从手机发出。

  2. 连usb，同时打开车机，正常充电，放音乐也正常，导航语音也从手机发出。

  3. 在2的情况下，如果同时打开蓝牙，则充电正常，但音乐与导航会出现干扰，就是如果用USB放音乐，音乐正常播放，但手机端导航会无声。如果用蓝牙放音乐，则音乐会在导航播报时暂停，但导航语音却无法听到。

  4. 断开USB，只用蓝牙连接手机与车机，则依然会出现音乐与导航语音不兼容情况。

  5. 开蓝牙时，将手机与车机的uconnect live连接上，用酷我音乐放音乐，则音乐正常播放，导航语音从车载喇叭发声。互不干扰。

  6. 在5的基础上，连上USB正常充电。没有问题。

  • 总结:

    1. 出现导航语音与音乐干扰情况，是在蓝牙连接时出现，如果又想充电，又想听导航，请关闭手机上的蓝牙连接。

    2. USB单连时，歌曲从车机发声，导航从手机发声，不干扰。

    3. USB充电任何时候都正常。

    4. 开蓝牙连接时，只有通过与uconnect live连接，才能使音乐与导航语音共同从车载喇叭发声。
